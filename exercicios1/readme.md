## Atividades do slide 1

1) Hello, World!: Escreva um programa que imprima "Hello, World!" na tela.
1) Calculadora Simples: Crie um programa que leia dois números do usuário e imprima a soma, subtração, multiplicação e divisão desses números.
3) Maior de Três Números: Escreva um programa que leia três números do usuário e imprima o maior deles.
4) Par ou Ímpar: Escreva um programa que leia um número inteiro do usuário e informe se ele é par ou ímpar.
5) Fatorial de um Número: Escreva um programa que calcule o fatorial de um número inteiro fornecido pelo usuário.
6) Números Primos: Crie um programa que verifique se um número fornecido pelo usuário é primo ou não.
7) Soma dos Dígitos: Escreva um programa que leia um número inteiro e calcule a soma de seus dígitos.
8) Sequência de Fibonacci: Escreva um programa que gere a sequência de Fibonacci até um número n fornecido pelo usuário.
9) Ordenação de Vetor: Crie um programa que leia um vetor de 10 números inteiros e o ordene em ordem crescente.
10) Palíndromo: Escreva um programa que verifique se uma string fornecida pelo usuário é um palíndromo ou não.
